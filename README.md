# Descrição: API que cadastra anuncios e fornece relatorios estatistico de alcance de visualizações


**Como buildar o projeto**

`mvn clean install
`

**Como acessar o swagger**

http://localhost:8080/swagger-ui.html

Java 11

SpringBoot

Spring devtoos

SpringData JPA

Swagger

H2 Database
